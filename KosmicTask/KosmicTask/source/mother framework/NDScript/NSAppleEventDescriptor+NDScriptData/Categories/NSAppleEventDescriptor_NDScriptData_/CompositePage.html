descriptorWithAEDescNoCopy:

Returns a NSAppleEventDescriptor for the AEDesc. 

+ (NSAppleEventDescriptor *)descriptorWithAEDescNoCopy:(const AEDesc *)aeDesc;
Parameters
aeDesc
A Carbon AEDesc structure.
Return Value

A NSAppleEventDescriptor

Discussion

Allocates, initializes and returns an NSAppleEventDescriptor that takes ownership of the Carbon AEDesc structure pointed to by aeDesc. Returns nil if an error occurs. The initialized object takes responsibility for calling the AEDisposeDesc function on the AEDesc at object deallocation time.

isTargetCurrentProcess

Determines if target is current process. 

- (BOOL)isTargetCurrentProcess;
Return Value

Returns YES if the recevier is an AppleEvent for the current process.

Discussion

If the recevier is a AppleEvent that contains a target ProcessSerialNumber that is the current process (ie you application) then this method returns YES.
