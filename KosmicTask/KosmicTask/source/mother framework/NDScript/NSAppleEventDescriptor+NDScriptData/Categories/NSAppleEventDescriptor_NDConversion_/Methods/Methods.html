Discussion

Returns a record descriptor. Any keys that are not NSNumber<unsigned long> are converted to NSStrings by there method -desccription and placed withing a list descriptor with there value, the array is then inserted into the record decriptor with the key keyASUserRecordFields; any keys that are NSNumber<unsigned long> are converted to AEKeyword with its method -unsignedIntValue. This is how record types in AppleScript are represented using AppleEvents
