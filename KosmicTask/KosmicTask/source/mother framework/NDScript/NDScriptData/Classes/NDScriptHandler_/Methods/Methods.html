initWithSource:modeFlags:componentInstance:

Initialize a NDSciptData 

- (id)initWithSource:(NSString *)source modeFlags:(long)modeFlags 
        componentInstance:(NDComponentInstance *)componentInstance;
Parameters
source
A NSString to be compilied.
modeFlags
Or'ed combination of values to determine how the script behaves whiles running.
componentInstance
The component instance to use, a value of nil to use the default AppleScript context.
Return Value

A initialized NDScriptHandler, or nil of initialization fails.

Discussion

Initializes the receiver with with compiled source, the modeFlags can be used to control ho the compilied script behaves when running and can be an or'ed combination of any of the following values;

kOSAModePreventGetSource
This mode flag may be used to instruct the scripting component to not retain the "source" of an expression. This will cause the method source to return the nil if used. However, some scripting components may not retain the source anyway. This is mainly used when either space efficiency is desired, or a script is to be "locked" so that its implementation may not be viewed.
kOSAModeNeverInteract
This mode flag indicates the script may interact with the user if necessary. Adds kAENeverInteract to the sendMode parameter of AESend for events sent when the script is executed.
kOSAModeCanInteract
This mode flag indicates the script may interact with the user. Adds kAECanInteract to the sendMode parameter of AESend for events sent when the script is executed.
kOSAModeAlwaysInteract
This mode flag indicates that the script may interact with the user. Adds kAEAlwaysInteract to the sendMode parameter of AESend for events sent when the script is executed.
kOSAModeDontReconnect
This mode flag indicates that the script may not reconnect if necessary. Adds kAEDontReconnect to the sendMode parameter of AESend for events sent when the script is executed.
kOSAModeCantSwitchLayer
This mode flag indicates whether AppleEvents should be sent with the kAECanSwitchLayer mode flag sent. This flag is exactly the opposite of the AppleEvent flag kAECanSwitchLayer. This is to provide a more convenient default, such as not supplying any mode (kOSANullMode) means to send events with kAECanSwitchLayer. Supplying the kOSAModeCantSwitchLayer mode flag will cause AESend to be called without kAECanSwitchLayer.

resultScriptData

Get the execution result. 

- (NDScriptData *)resultScriptData;
Return Value

A NDScriptData representing the result.

Discussion

Returns NDScriptData or a subclass for the result of the last execution of the reciever. If the result type is a script then resultScriptData returns a NDScriptContext instance, if the result is a script handler then resultScriptData returns a NDScriptHandler instance, for all other results types resultScriptData returns a NDSciptData.

source

Retrieve script source. 

- (NSString *)source;
Return Value

Returns a NSString contain the source if available; otherwise nil is returned.

Discussion

source decompiles the reciever if available and returns it, if the source is not available then nil is return.
