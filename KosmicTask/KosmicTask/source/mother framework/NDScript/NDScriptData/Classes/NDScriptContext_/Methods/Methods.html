appleEventTarget

Get an AppleEvent desriptor that can be used in constructing complete AppleEvents. 

- (NSAppleEventDescriptor *)appleEventTarget;
Return Value

A NSAppleEventDescriptor to be used as an AppleEvent target.

Discussion

When construction AppleEvents using the NSAppleEventDescriptor method appleEventWithEventClass:eventID:targetDescriptor:returnID:transactionID: to send to the script with the NDScriptData method executeEvent: the NSAppleEventDescriptor return from this method can be used as the target discriptor.

scriptDataForPropertyNamed:

Return the data for a given property name. 

- (NDScriptData *)scriptDataForPropertyNamed:(NSString *)propertyName;
Parameters
propertyName
The property name.
Return Value

The script data or nil if no data exists.

Discussion

scriptDataForPropertyCode: returns the data for a given name propertyName. Some properties are not represented by there name but instead use a code, for example parent or name can not be returned with the strings "parent" or "name". Instead the codes 'pare' or 'pnam' must used with the method scriptDataForPropertyCode:. This can be changed in the script source by enclosing the property name with '|' character for example |parent| or |name|.
