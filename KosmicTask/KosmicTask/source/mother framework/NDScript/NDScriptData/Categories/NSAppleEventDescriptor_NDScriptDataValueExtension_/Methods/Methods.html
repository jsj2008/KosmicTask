descriptorWithScriptData:

Create a AppleEvent decsriptor with script data. 

+ (NSAppleEventDescriptor *)descriptorWithScriptData:(NDScriptData *)scriptData;
Parameters
scriptData
The script data to create the AppleEvent descriptor.
Return Value

A new NSAppleEventDescriptor, or nil if the method fails.

Discussion

To pass script data in AppleEvents it first has to be converted to a AppleEvent descriptor, converting script data to Cocoa type also requires converting the data to a AppleEvent descriptor first, this function does this but you can also just as easily use the method -[NDScriptData appleEventDescriptorValue].

scriptDataValue

Category method for NSAppleEventDescriptor (NDScriptContextValueExtension), converts any script data within a AppleEvent descriptor into an NDScriptContext 

- (NDScriptData *)scriptDataValue;
Return Value

a NDScriptContext object for the AppleScript contained within the AppleEvent descriptor.

Discussion

If an AppleScript return a AppleScript as it’s result, this method can be used to convert the result NSAppleEventDescriptor into a NDScriptContext. The NSAppleEventDescriptor method objectValue will use this method if available.
