executionModeAlwaysInteract

. 

- (BOOL)executionModeAlwaysInteract;
Return Value

Returns YES if AlwaysInteract; otherwise returns NO.

Discussion

Indicate whether or not the script should always interact with the user. Adds kAEAlwaysInteract to the sendMode parameter of AESend for events sent when the script is executed. This is equivalent to ([self executionModeFlags] & kOSAModeAlwaysInteract)

executionModeRecord

. 

- (BOOL)executionModeRecord;
Return Value

Returns YES if record; otherwise returns NO.

Discussion

Indicate whether AppleEvents should be sent with the kAEDontRecord mode flag. This flag is exactly the opposite the AppleEvent flag kAEDontRecord. This is to provide a more convenient default, such as not supplying any mode (kOSANullMode) means to send events with kAEDontRecord. Supplying the kOSAModeDoRecord mode flag will cause AESend to be called without kAEDontRecord. This is equivalent to ([self executionModeFlags] & kOSAModeDoRecord)
