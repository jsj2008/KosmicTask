NDProgrammerUtilities

Collection of function to help with developement and testing.

Includes: <Carbon/Carbon.h>, <Cocoa/Cocoa.h>
Discussion

These function will by default display a message when an error condition occurs, this can be disabled by defining the macro varible NDTurnLoggingOff. Errors condition can be made to throw an NSInternalInconsistencyException by defining the macro varible NDAssertLogging.


#defines

NDLogFalse 

#define NDLogFalse( CONDITION_ )
Parameters
CONDITION_
An expression that can is evaluated as YES or NO, evaluated only once. The macro varible NDTurnLoggingOff can be defined to turn off this message. The NSInternalInconsistencyException can be thrown instead by defining the macro varible NDAssertLogging.
Return Value

if CONDITION_ evaluates to YES then NDLogFalseBody returns YES otherwise it return NO.

NDLogOSStatus 

#define NDLogOSStatus( OS_ERROR_ )
Parameters
OS_ERROR_

Return Value
