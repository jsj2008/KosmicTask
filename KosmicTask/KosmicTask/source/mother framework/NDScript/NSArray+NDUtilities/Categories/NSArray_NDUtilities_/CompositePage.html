arrayByUsingFunction:

Returns an array of the result a function passed every object. 

- (NSArray *)arrayByUsingFunction:(id (*)(id, BOOL *))func;
Parameters
func
The function pointer.
Return Value

The results NSArray

Discussion

Every object in the receiver is passed to the supplied function and all of the none nil results are added to a new NSArray. A pointer to a BOOL is also passed to the function and if the value it points to is change to NO by then function then the enumeration stops. The function should be of the form

id func( id anObject, BOOL * aFlag)
Name Description
func The function name.
anObject The object passed to the function.
aFlag A pointer to a BOOL used to stop enumeration over all of the object in the NSArray.
Result: The object to add to the new array or nil to add no object.

objectEnumeratorWithIndicies:

Returns an enumerator to enumerate over the objects with a NSArray as determined by the indices within indicies. 

- (NSEnumerator *)objectEnumeratorWithIndicies:(NSIndexSet *)indicies;
Parameters
indicies
A NSIndexSet containing the indexes to enumerate over.
Return Value

An NSEnumerator.

Discussion

Returns an enumerator object that lets you access each object in the receiver coresponding to the indicies within indicies, in order, starting with the element coresponding to the first index in indicies, as in: 

 NSEnumerator *theEnumerator = [myArray objectEnumeratorWithIndicies:anIndicies];
 id theObject;

 while (theObject = [theEnumerator nextObject])
 {
 // code to act on each element as it is returned
 }

This is equivelent to; 

 NSEnumerator *theEnumerator = [[array objectsAtIndexes:indicies] objectEnumerator];
 id theObject;

 while (theObject = [theEnumerator nextObject])
 {
 // code to act on each element as it is returned
 }

When this method is used with mutable subclasses of NSArray, your code shouldn’t modify the array during enumeration. If indicies is nil then the result of the method objectEnumerator is returned instead.

sendEveryObjectToTarget:withSelector:

Pass contents to an object. 

- (void)sendEveryObjectToTarget:(id)target withSelector:(SEL)selector;
Parameters
target
The object each object in the reciever is sent to.
selector
The selector for the message sent to target
Discussion

sendEveryObjectToTarget:withSelector: passes every object within the reciever to the object target one at a time through the selector selector. The returned value of the selector selector is ignored.
